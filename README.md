
# Student Marks Predictor

A simple Machine Learning web application built using Flask that predicts student marks based on study hours.

---

## 📌 Project Overview

This project uses a Simple Linear Regression Machine Learning model to predict student marks from the number of study hours entered by the user.

The application is developed using:

- Python
- Flask
- Scikit-learn
- HTML
- CSS

---

## 🚀 Features

- Predict student marks
- Simple and beginner-friendly UI
- Flask web application
- Machine Learning integration
- Linear Regression model
- Real-time prediction output
